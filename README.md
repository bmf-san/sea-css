# Customizing
sea.css uses a basic set of css variables, so you can override the variables defined in sea.css.

```css
@import url(sea.css)

:root {
  --main-text:Helvetica;
  --sub-text:Futura;
  --main-color: #21ADC7; 
  --footer-color: #1F8699; 
  --maintext-color: #262424; 
  --subpoint-color: #647172; 
  --border-color:#C7DADD;
}
```
